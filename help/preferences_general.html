General Preferences
Minimize to Tray When enabled and the system tray is avaialble this will force Shubetria to minimize to the tray instead of the task bar.
Always Show Tray Icon When enabled and the system tray is available this will alway display the Shubetria icon in the system tray.
Start Minimized When enabled this will automatically minimize Shubetria when launched.
Quit on Close When disabled Shubetria will always remain running and will minimize or hide itself instead of exiting.
Startup Profile Shubetria will attempt to load this profile when starting.
Shutdown Profile Shubetria will attempt to load this profile when shutdown.
