File Menu
Load Profile This is used to quickly change to a custom profile.
Preferences This allows you to change the behavior and appearance of the Shubetria.
Quit This allows you to change the QUIT behavior to prevent Shubetria from shutting down.
Help Menu
Diagnostic Report A basic environmental report of your hardware and software configuration for support purposes.
About Software and license information as well as notable acknowledgements for the people who have provided to the development of Shubetria.
Application Icon and Current Profile
Application Icon This icon will turn red if the Recon isn't detected or blue the connection is successful.
Current Profile This displays the profile name that is currently being used.
Icon Buttons
Temperature Toggle This button toggle's the temperature displayed between celsius and fahrenheit.
Alarm Toggle This button toggle's the alarm and audible key presses of the Recon on and off.
Manual Mode Toggle This button toggle's the Recon's automatic and manual modes.
Profile Configuration This is used to configure profiles to be used while the software is running.
Auto Mode Toggle This toggle's the advanced software auto features (fan curves) on and off, this feature is only available while Shubetria is running.
Software Auto Configuration This button is used to customize the advanced software auto (fan curve) profiles.
Channel Controls
Current Temperature This displays the current temperature; it may also display a smaller number if probe affinity is enabled.
RPM Slider This manually controls the rpm of your fans in increments of 100, the logarithmic scale can be disabled through preferences.
RPM Indicator This displays the current rpm of the fan, the indicator will "slide" as the rpm increases or decreases.
Current RPM This is the numerical value of the current rpm.
Channel Icon This button is used to change the channel on the Recon display.
Alarm Temperature This button will display and set the maximum temperature allowed before an audible alarm will sound.
