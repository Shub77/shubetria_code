Channel Settings
Channel Select the channel you would like to define a fan curve for.
Minimum settable RPM Allows you to control the minimum rpm that can be set for that channel.
Turn fan on at The temperature the probe must achieve before it will turn on the fan to the specified rpm.
Always on Forces the fan to stay on at all times.
Ramp start The temperature and speed where the fan will start it's ramp.
Ramp mid The temperature and speed where the fan will reach the mid point of its ramp.
Ramp end The temperature and speed where the fan will reach the end of its ramp.
Fan to max RPM The temperature at which the fan will enable its maximum rpm.
Advanced
Hysteresis Up The number of degree's the temperature probe has to increase above the current setting before rpm will be increased.
Hysteresis Down The number of degree's the temperature probe has to decrease bellow the current setting before rpm will be decreased.
