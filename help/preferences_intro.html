General Information
Preferences Preferences are stored in your configuration folder in a file named Phoebetria.ini, it's a standard text file readable by any text editor.
Profiles Profiles are stored in your configuration folder in a file named Phoebetria.sqlite, this is a sqlite database and is only viewable by the appropriate software.
Themes Custom themes are stored in your configuration\themes folder, themes will automatically be added to the themes menu if they have ".qss" for their extension.
Configuration Folders

In the following paths please replace {username} with your own user name
Windows XP C:\Documents and Settings\{username}\Application Data\Phoebetria\
Windows 7 and higher C:\Users\{username}\AppData\Roaming\Phoebetria\
Linux (Fedora and Ubuntu) ~/.config/Phoebetria/ or /home/{username}/.config/Phoebetria/
OS X SnowLeopard and higher ~/.config/Phoebetria/ or /home/{username}/.config/Phoebetria/
